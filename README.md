# Report Export Service

Сервис для анализа текстовых файлов и генерации Excel-отчёта со
статистикой словоформ.

------------------------------------------------------------------------

## Запуск

### 1. Клонировать проект

git clone `<repo_url>`

### 2. Запустить через Docker

docker-compose up --build

------------------------------------------------------------------------

## Доступ

Так как интерфейса не предусмотрено, будем использовать встроенный в FastAPI
-   Swagger UI: http://localhost:8000/docs

------------------------------------------------------------------------

## Как протестировать

### 1. Загрузка файла

-   Открыть http://localhost:8000/docs
-   Найти: POST /public/report/export
-   Нажать Try it out
-   Загрузить .txt файл
-   Выполнить запрос (Execute)

В ответе будет: { "task_id": "...", "status": "processing" }

------------------------------------------------------------------------

### 2. Проверка статуса

-   Найти: GET /tasks/{task_id}
-   Вставить task_id из первого шага
-   Выполнить запрос (Execute)

Убедиться что нам вернулось: { "status": "SUCCESS" }

------------------------------------------------------------------------

### 3. Скачать результат

-   Найти: GET /tasks/{task_id}/download
-   Вставить task_id из первого шага
-   Выполнить запрос (Execute)

Скачается .xlsx файл

------------------------------------------------------------------------

## Особенности

-   Обработка больших файлов (стриминг)
-   Лемматизация слов (pymorphy2)
-   Асинхронная обработка (Celery + Redis)
-   Генерация Excel (xlsx)
